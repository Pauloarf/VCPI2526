# VCPI 2025/2026
This repository hosts projects developed for the **Vision and Image Processing (VCPI)** course at the **University of Minho (UM)**. The curriculum is divided into two core domains: Image Processing (PI) and Computer Vision (VC).

## 🚦 Computer Vision (VC) – Traffic Sign Recognition
**Objective:** Developed a robust deep learning model to classify traffic signs using the **GTSRB (German Traffic Sign Recognition Benchmark)** dataset.

* **Key Features:**
    * Implemented **Data Augmentation** (rotation, scaling, brightness adjustment) to improve model generalization.
    * Explored various Neural Network architectures for optimal accuracy.
    * **Tools:** Python, PyTorch/TensorFlow, OpenCV.



## 🔍 Image Processing (PI) – Hidden Letter Extraction
**Objective:** Applied "classic" image processing pipelines to identify and isolate hidden text within complex or noisy backgrounds.

* **Techniques Implemented:**
    * **Spatial Filtering:** Noise reduction using median and Gaussian filters.
    * **Spectral Filtering:** Frequency domain analysis (Fourier Transform) to remove periodic noise.
    * **Segmentation:** Thresholding, edge detection (Canny, Sobel), and morphological operations to isolate letter contours.
* **Tools:** Python (OpenCV & NumPy).
